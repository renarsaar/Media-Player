## General info

This is a Media/Music Player made using React.js. The main purpose of this project was to work with different API's and practice React/Redux.

## Technologies

Project is created with:

- React
- Redux
- Sass
- Youtube Data v3 Api
- Deezer Music Api

## File Structure:

File structure of this project. Data is coming from Redux store

```
|-- src
|   |-- actions
|       |-- index.js
|       |-- types.js
|   |-- apis
|       |-- deezer.js
|       |-- videos.js
|   |-- components
|       |-- App.js
|       |-- MusicPlayer.js
|       |-- SearchBar.js
|       |-- VideoDetail.js
|       |-- VideoItem.js
|       |-- VideoList.js
|       |-- VideoPlayer.js
|   |-- reducers
|       |-- index.js
|       |-- songsReducers.js
|       |-- videosReducers.js
|   |-- index.js
```

The Project is live at: [vra.ee/player](https://vra.ee/player/)
